🎓 Features

  • Real-time pose detection - Get joint angles every frame
  • Automatic recording - Collect N seconds of data
  • Statistics - Get min/max/avg/std for each joint
  • API integration - Send data directly to Django backend
  • Data export - Download JSON for analysis
